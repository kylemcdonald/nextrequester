# nextrequester

A hacky bit of python and selenium to navigate the atrociously designed NextRequest public records request websites.

## Requirements
- Linux or OSX
- Python 3.7 or higher
- Pip
- whatever's in the `requirements.txt` file

## Installation

1. Clone the repo
  ```shell script
git clone 
```
1. Install requirements
1. Set config values
